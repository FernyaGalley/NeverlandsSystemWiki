There are game mechanics in the Neverland System that Players are likely to engage with through the course of play.

**Narrative Elements** are physical structures that can be interacted with by the Players. In the [[Neverlands System]], the word Element (short for Narrative Element) is always capitalized, as are all Elements, to help players easily identify Elements by sight. In this particular version of the Neverlands System, the first time an Element is referenced in an article it will also have a link to the page describing what that Element is and how to interpret Statements with those Elements.

**Rules** are metaphysical properties that explain how Elements behave under different scenarios. In the Neverlands System, Rules are not capitalized, but they are meta-narrative Statements about different Elements and the recommended way to play with those Elements. Rules are not etched in stone; if a Director determines that a Rule would be uncomfortable to implement or harmful to Players for any reason, they are encouraged to bend and rewrite Rules. As a general rule (pun intended), Rules should make the Neverland System easier to comprehend and allow Players to resolve conflicts in a timely and effective manner.

*What follows* is a list of each category of Narrative Element or Rule that exists within the Neverland System. Directors are encouraged to understand the existing Narrative Elements and Rules before finding new ones. These categories are rather broad and useful for all kinds of stories, and should only require minimal adaptation to fit the Table's needs.

Players who struggle to understand the following index are encouraged to visit the [[Appendix]] and read the section on Entry-Level Open Source Textbooks. There's no shame in needing to learn basic logic, language arts or storytelling techniques before playing.

# Index of Game Mechanics (A-Z)

[[Arbitration]]
Arbitration is the mechanism by which the Director determines a Statement's kind or veracity.

[[Statements]]
Statements are phrases that can be logically determined as Fact, Fiction, Opinion or a Question.

[[Positions]]
A Position is a Statement which puts forward a claim to Fact and can be validated through logic.

[[Cards]]
Cards include any element which contain or display Statements and can be placed at a Position.
